# Dotfiles for bspwm

## terminal emulator
```
kitty
```
## program launcher
```
rofi
```

## other programs 
```
nitrogen - wallpapper setter
maim - screenshot tool
playerctl - utility for controlling media players 
brightnessctl - utility for controlling brightness
pamixer - utility for controlling volume
```
